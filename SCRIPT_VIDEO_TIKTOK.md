## 🎬 SCRIPT COMPLET — Scène par scène

---

### ⚡ HOOK [0 — 3 secondes]
**L'objectif : arrêter le scroll en 1 seconde.**

**Visuel :**
Fond noir total. Silence 0,5s.
Puis texte ÉNORME qui claque à l'écran, blanc, ultra bold :

```
"Ton app de fitness
te coach pas vraiment."
```

Animation : chaque ligne arrive comme un coup de poing — slide depuis la gauche, impact frame (flash blanc 1 frame), puis settle.

**Son :** début brutal de la musique exactement au premier mot. Pas de fade in.

**CapCut :** Texte > Police "Impact" ou "Bebas Neue" > Taille max > Animation entrée "Bounce" 0.2s

---

### 🌑 TRANSITION [3 — 4s]
Fond noir → ambient glow orange qui pulse depuis le centre.  
Couleur : `#FF6B35` très diffuse, comme un soleil qui se lève.

Le téléphone apparaît en 3D depuis le fond, rotation Y de 45° → 0°, lent et fluide (0.8s).

**CapCut :** Effet > 3D > "Rise" ou utilise un template "Phone reveal"

---

### 📱 SCÈNE 1 — Dashboard [4 — 10s]
**Vibe : "Vue d'ensemble. Tout en un coup d'œil."**

Le téléphone est centré, légère inclinaison 3D (perspective). Il oscille très doucement (comme posé sur une surface, effet flottant).

Screen recording du Dashboard qui scroll lentement vers le bas.

**Texte overlay (côté gauche, apparaît mot par mot) :**
```
Ligne 1 : "Tes séances"      [apparaît à 4.5s]
Ligne 2 : "Tes calories"     [apparaît à 5.5s]
Ligne 3 : "Ton streak"       [apparaît à 6.5s]
Ligne 4 : "Tout. Là."        [apparaît à 7.5s — plus gros, orange]
```

**Transition sortie :** le téléphone pivote sur l'axe Y (comme une page qui se tourne), 0.4s, puis la scène suivante arrive par derrière.

---

### 💪 SCÈNE 2 — Nouvelle séance [10 — 18s]
**Vibe : "Simple. Rapide. Efficace."**

**Split en 3 temps dans cette scène :**

**[10-12s] — Choix du type de séance**
Zoom doux sur l'écran pendant que le doigt tape "PUSH".
Le bouton orange s'allume.
Texte flash en haut : `"Push. Pull. Legs."` en orange, disparaît vite.

**[12-15s] — Ajout des exercices**
Screen recording accéléré ×1.5 — tu ajoutes Bench Press, puis les séries apparaissent.
Chaque exercice qui s'ajoute : micro flash orange sur l'écran.

Texte côté droit, petit, qui s'accumule comme un log :
```
+ Bench Press          [12.5s]
+ Incline Dumbbell     [13.2s]
+ Cable Fly            [14s]
```

**[15-18s] — Une série complétée**
Zoom in sur les séries (poids × reps). Doigt qui tape pour valider.

Texte gros au centre, impact :
```
"100kg × 5"
```
Reste 1 seconde, puis...
**Confetti orange** qui tombe (CapCut > Stickers > confetti).

---

### 🤖 SCÈNE 3 — AI Coach [18 — 32s]
**C'est LA scène clé. 14 secondes dessus. Le produit se vend ici.**

**[18-19s] — Transition vers AI Coach**
Le téléphone fait une rotation 3D complète sur 360° (effet "flip").
Pendant la rotation : le screen change de Dashboard → AI Coach.
Son : petit whoosh.

**[19-22s] — Le problème posé**
L'écran montre le chat vide.
Texte overlay, style "machine à écrire" :
```
"Et si ton coach
connaissait tous tes PRs ?"
```

**[22-26s] — La question au coach**
Screen recording en temps réel : tu tapes lentement dans le chat.

Message visible à l'écran :
```
"Crée-moi une séance Push
pour demain, j'ai mal à l'épaule droite"
```

Pendant que tu tapes : texte overlay en petit en haut :
```
"Il a accès à TOUT ton historique."
```

**[26-32s] — La réponse de l'IA**
La réponse du coach apparaît (screen recording).
Pendant que le texte scroll :

Texte overlay qui apparaît sur les côtés, un par un comme des bullets :
```
✓ Il connaît tes PRs         [27s]
✓ Il mémorise tes blessures  [28.5s]
✓ Il crée les séances        [30s]
✓ Il suit tes calories       [31.5s]
```

Chaque bullet : slide depuis la droite, couleur verte/orange alternée.

---

### 📊 SCÈNE 4 — Analytics [32 — 38s]
**Vibe : "La data. Tes progrès. Visuellement."**

**[32-33s] — Transition**
Zoom out rapide du téléphone → le téléphone rétrécit au centre → zoom in sur Analytics.
Effet "tunnel" avec les tabs qui défilent.

**[33-38s] — Scroll rapide des 4 tabs**
Screen recording accéléré ×2 — tu scrolles dans Analytics, tu changes les tabs.

4 textes flashent rapidement (chacun 1 seconde) :
```
[33s] "Tes records" → tab Muscu avec graphe
[34s] "Ta distance" → tab Cardio
[35s] "Ton poids" → tab Poids avec courbe
[36s] "Tes posings" → tab Photos galerie
```

Chaque texte : couleur différente, apparaît et disparaît vite (style slideshow).

Le tout avec le téléphone légèrement incliné en 3D, effect parallax.

---

### 📷 SCÈNE 5 — Photos posing [38 — 43s]
**Vibe : "Tes progrès. En images."**

**[38-40s]**
La galerie photos en plein écran (screen recording).
Zoom in lent sur une photo de la galerie.

Texte overlay en bas, style caption Instagram :
```
"Photo après chaque séance.
Stockée dans le cloud. 📱→☁️"
```

**[40-43s]**
Zoom out → on voit la grille 3×3 de photos.

Texte gros, center screen, blanc :
```
"Ton évolution.
Mois après mois."
```

---

### 🔗 SCÈNE 6 — Intégrations [43 — 48s]
**Rapide. Impact. Pas besoin de s'étendre.**

**[43-44s] — Transition slam**
Cut brutal (pas de transition douce ici, intentionnel).

**[44-48s]**
Screen recording de Paramètres — badges Strava vert + Garmin vert.

Texte overlay très rapide, style "tech spec" :
```
⚡ Strava connecté
⚡ Garmin connecté
⚡ Calories importées auto
⚡ Aucune saisie manuelle
```

Chaque ligne : arrive comme une notification, 0.4s d'intervalle.

Fond : léger glow vert derrière le téléphone pour matcher les badges verts.

---

### 🌑 SCÈNE 7 — Silence avant le CTA [48 — 50s]
**2 secondes de pause.**

Le téléphone disparaît en fondu.
Fond noir total.
Silence musical (ou beat qui se coupe).

Juste ce texte, petit, centré, blanc :
```
"Disponible maintenant."
```

Puis ça grossit lentement.

---

### 🔥 SCÈNE 8 — CTA FINAL [50 — 60s]
**C'est là que tu demandes l'action.**

**[50-52s] — Badge App Store**
Animation : badge "Bientôt sur App Store" qui arrive depuis le haut, rebondit, se stabilise.
Couleur : blanc sur fond noir.

**[52-56s] — Texte principal**
3D text reveal — les lettres arrivent une par une, style cinématique :

```
"PRÉ-INSCRIPTION"
"GRATUITE"
```

Police très bold, très grande, gradient orange → rouge.

**[56-58s] — URL**
```
trackermuscu.fr
```
Texte plus petit, blanc, légère lueur orange derrière.

**[58-60s] — Call to action final**
```
"👆 LIEN EN BIO"
```
Texte orange, clignote 2 fois, reste affiché.

Musique : monte légèrement pour finir sur un pic.

## 🎵 MUSIQUE

**Style recherché :** dark phonk / gym motivation / cinematic drill

**Cherche sur :**
- TikTok Sound Library (intégré dans CapCut) → "dark phonk"
- YouTube Audio Library → "workout motivation"
- Suggestions : "Monkeys Spinning Monkeys" type beat, ou phonk sombre

**Structure musicale :**
- Intro : beat minimal (hook)
- Drop au moment où le téléphone apparaît (4s)
- Break (ralentit ou s'arrête) au moment du silence CTA (48s)
- Remonte pour le CTA final

## 📱 COMMENT FAIRE L'EFFET 3D TÉLÉPHONE (sans budget)

### Option A — ScreenStudio (le plus simple)
1. Télécharge ScreenStudio (Mac uniquement, trial gratuit)
2. Lance l'enregistrement de Chrome en mode iPhone
3. ScreenStudio ajoute automatiquement le cadre iPhone, l'ombre, la rotation 3D
4. Exporte → importe dans CapCut

### Option B — CapCut seul (gratuit)
1. Screen record l'app dans Chrome DevTools iPhone mode
2. Dans CapCut : Effets > Cadre > "Phone Frame" → choisit iPhone 14 Pro
3. Applique l'effet "3D Parallax" sur le clip
4. Pour la rotation : Keyframes → position X au début -30°, à la fin 0°

### Option C — Rotato (le plus beau)
1. rotato.app → upload tes screenshots
2. Crée des animations 3D du téléphone avec tes screens dedans
3. Export en vidéo → importe dans CapCut pour le montage
