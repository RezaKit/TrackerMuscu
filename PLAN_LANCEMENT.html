📱 Phase 1 — Screenshots & Assets

Durée estimée : 1-2h · À faire en premier

Comment faire les screenshots iPhone

  1. Ouvre Chrome sur ton Mac
  2. F12 → icône mobile (DevTools) → sélectionne "iPhone 14 Pro"
  3. Va sur tracker-muscu-xi.vercel.app
  4. Connecte-toi avec ton compte, mets de vraies données
  5. Fais des captures d'écran (Cmd+Shift+4)

Écrans à capturer (minimum 15)

  • Dashboard avec données remplies (streak, calories, séances)
  • Nouvelle séance — type picker (les 5 types colorés)
  • Séance en cours avec exercices + séries
  • AI Coach — une vraie conversation avec réponse du coach
  • AI Coach — confirmation d'action (créer une séance)
  • Analytics — onglet Muscu avec graphes
  • Analytics — onglet Cardio
  • Analytics — onglet Poids (courbe)
  • Analytics — onglet Photos (galerie posing)
  • Calendrier avec séances colorées
  • Détail d'un jour dans le calendrier
  • Paramètres — Strava connecté (badge vert)
  • Paramètres — Garmin connecté
  • Post-session photo (modal posing)
  • Routine du soir avec items cochés

Screen recording (30-60 secondes)

  • Sur iPhone : Centre de contrôle → Enregistrement d'écran
  • Navigue naturellement : Dashboard → Nouvelle séance → AI Coach → Analytics
  • Fais-le 2-3 fois, garde le meilleur

🌐 Phase 2 — Landing page pré-inscription

Durée estimée : 3-4h

Outils

  • Carrd.co — 9€/an (le plus simple, parfait pour landing one-page)
  • Brevo.com — gratuit jusqu'à 300 emails/jour (collecte + envoi)

Structure de la landing page

  1. Hero — Titre choc + screenshot dashboard + CTA "Pré-inscription gratuite"
  2. Le problème — "T'as essayé 10 apps. Aucune t'a vraiment coaché."
  3. Features — 4 blocs avec screenshots : AI Coach / Tracking / Photos / Intégrations
  4. Bientôt sur App Store — badge Apple + date estimée
  5. Formulaire email — "Rejoins les X premiers" (connecté à Brevo)
  6. Footer — Instagram + TikTok

Setup Brevo (emails)

  1. Crée compte sur brevo.com (email + mdp)
  2. Contacts → Listes → "Créer une liste" → nomme "Waitlist TrackerMuscu"
  3. Formulaires → Créer un formulaire → copie le code HTML
  4. Colle dans Carrd à la place du bouton CTA

🍎 Phase 5 — App Store (Capacitor)

Durée estimée : 1 semaine · Nécessite Mac + Xcode

ÉtapeActionTemps
1Compte Apple Developer (developer.apple.com) — 99€/an30min
2Installer Capacitor : npm install @capacitor/core @capacitor/ios1h
3Remplacer input camera par plugin Capacitor Camera2h
4Ajouter permissions iOS dans Info.plist30min
5Build + test sur simulateur Xcode2h
6Créer assets App Store (icônes, screenshots App Store)2h
7Soumettre via App Store Connect1h
8Review Apple (1-5 jours)attente

Changements requis dans le code

  • Remplacer <input type="file"> par @capacitor/camera pour les photos posing
  • Ajouter NSCameraUsageDescription et NSPhotoLibraryUsageDescription dans Info.plist
  • Activer les push notifications avec @capacitor/push-notifications
  • Intégrer Apple Health via capacitor-health-connect (optionnel mais fort argument)
  • Intégrer RevenueCat pour le paiement In-App Purchase
